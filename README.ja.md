# text-reverse
The Japanese README is here: [README.ja.md](README.ja.md)

簡単なテキスト反転ツール。

## 機能
- 入力テキストの行順序を逆順にする
- 複数行のテキスト入力をサポートする

## 使用方法
1. 上部のテキストエリアに反転させたいテキストを入力してください。
2. 反転したテキストは自動的に下部のテキストエリアに表示されます。
